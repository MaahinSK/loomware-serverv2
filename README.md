# LoomWare Server

Backend API for LoomWare Garments Order & Production Tracker System

## Features
- ✅ User Authentication (JWT + Firebase)
- ✅ Role-based Access Control (Buyer, Manager, Admin)
- ✅ Product Management
- ✅ Order Processing
- ✅ Payment Integration (Stripe)
- ✅ Order Tracking
- ✅ Image Upload (Cloudinary)
- ✅ Real-time Updates

## Tech Stack
- Node.js & Express.js
- MongoDB & Mongoose
- JWT Authentication
- Cloudinary for Image Storage
- Stripe for Payments
- Firebase for Auth
